# B-Ware E-Commerce

German e-commerce front-end for discounted home appliances (B-Ware) with strong defect transparency UX.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/src/app` - Next.js App Router pages
- `/src/components` - React components
- `/src/data` - Mock data
- `/src/lib` - Utilities and types
